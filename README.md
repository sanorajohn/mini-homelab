# Mini Homelab Project

## Overview
Built a small homelab focused on IT support and basic networking practice using VMware Workstation Pro. This environment is designed to strengthen troubleshooting skills, system administration fundamentals, and hands‑on networking experience.

## Hardware
- HP EliteDesk Mini PC
- Netgear WiFi Router
- TP‑Link 8‑Port Gigabit Switch

## Tools
- VMware Workstation Pro
- Windows 11 Pro
- Windows Server 2022
- Windows 10 (VM)

## What I'm Practicing
- Helpdesk fundamentals (OS installation, system configuration, remote administration)
- Basic networking (IP configuration, DNS testing, connectivity troubleshooting)
- Windows administration (user accounts, permissions, services)

## Virtual Machines
- Windows 10 – troubleshooting, OS practice, user environment issues
- Windows Server 2022 – Active Directory, user management, system configuration
